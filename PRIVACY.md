# Privacy Policy for Z80CPM

**Last updated:** December 25, 2025

## Overview

Z80CPM is a Z80 CPU emulator for running CP/M and vintage operating systems. This application is designed with privacy in mind.

## Data Collection

**Z80CPM does not collect, store, or transmit any personal information.**

Specifically, the application:
- Does not collect personal data
- Does not track usage or analytics
- Does not require user accounts
- Does not access the internet (except for optional disk image downloads from GitHub)
- Does not share any data with third parties

## Local Storage

The application stores the following data locally on your device only:
- User preferences (font size, window position) in a local configuration file
- Downloaded disk images in the application folder

This data never leaves your device.

## Optional Network Access

The only network feature is the optional disk catalog, which downloads publicly available disk images from GitHub. No personal information is transmitted during this process.
